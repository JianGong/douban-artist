# 使用的库

```
phonegap 3.3 
AngularJS 1.2.6
ionic 0.9.17
iScroll 5.0.9
```

# Build

请先按官网介绍安装好 phonegap 3.3 环境，然后 cd 到根目录，运行：
```sh
phonegap run android
```
如果一切顺利的话会在模拟器中看到运行的 app。

OS X 下的 build 没有尝试过，因为买不起 mac。
